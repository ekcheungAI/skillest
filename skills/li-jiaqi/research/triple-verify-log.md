# Triple Verification Log — 李佳琦 (Austin Li)

> 填充版本。从 candidate patterns 中筛选，最终进入 SKILL.md 的结果记录。

**Rule:** A pattern qualifies as a **Mental Model** only if it passes ALL THREE tests:
- Test 1: Cross-Domain Reproduction (≥2 different life domains)
- Test 2: Generative Power (predict novel stance not directly quoted)
- Test 3: Non-Obvious / Exclusive (distinctively theirs, not universal wisdom)
